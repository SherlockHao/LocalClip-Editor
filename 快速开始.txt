====================================
  LocalClip Editor - 快速开始
====================================

📌 双击启动
------------------
双击 "启动 LocalClip Editor.vbs"

等待 10-15 秒后，浏览器会自动打开


📌 服务地址
------------------
任务看板: http://localhost:5173/dashboard
后端 API: http://localhost:8080/docs


📌 使用流程
------------------
1. 点击"创建新任务" → 上传视频和字幕
2. 点击任务卡片进入编辑器
3. 说话人识别 → 翻译 → 语音克隆 → 导出


📌 停止服务
------------------
双击 "停止 LocalClip Editor.vbs"


📌 支持的语言
------------------
✓ 英语 (English)
✓ 韩语 (Korean)
✓ 日语 (Japanese)
✓ 法语 (French)
✓ 德语 (German)
✓ 西班牙语 (Spanish)
✓ 印尼语 (Indonesian)


📌 遇到问题？
------------------
1. 查看 "使用说明.md" 中的故障排查
2. 检查后端窗口的日志输出
3. 确保 Python、Node.js、Ollama 已安装


====================================
祝您使用愉快！
